================
台大選課輔助程式
================

這是用來架設幫台大學生選課所用的網站程式： http://ntucourse.ericyu.org/

本程式中有用到一些其他人所寫的程式，在個別程式應該都有註明其授權，譬如 phpBarGraph2.php, color.js 等。

其他部分的程式主要貢獻者：ericyu, kcwu, piaip, tkirby。

授權
====
除了上面提到的那些不是我們寫的程式，其他的部分簡單來講就是歡迎拿去做任何用途。如果用此整套程式架設網站，不論是不是要將原始碼開放，都應在網頁上註明貢獻者，並連至此網站： https://bitbucket.org/ericyu/ntucourse
